# 二遊事件更新日曆 (Hoyo Calendar)

一個專為 HoYoverse 遊戲（以及《鳴潮》）打造的活動日曆工具。  
可以快速查看版本更新時間，並一鍵同步到 Google 日曆。

---

## 功能

- 顯示遊戲事件日曆（支援 原神 / 鐵道 / 鳴潮）
- 一鍵更新最新活動（自動抓取官方資料）
- 導出事件到 Google 日曆
- 一鍵刪除由本網站建立的事件
